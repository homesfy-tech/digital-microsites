## File Format

### CSV Export
- Properly escaped CSV format with headers
- Compatible with Excel, Google Sheets, and all spreadsheet applications
- Format: `Classname (Key),Content`

### Word Document Export
- Professional HTML-based Word document
- Includes all sections with proper formatting
- Tables for pricing information
- Lists for offers, highlights, and locations

## Sections Included

- **Basic Information**: Project name, developer name
- **SEO & Meta**: Meta title, description, keywords
- **Banner Information**: Title, subtitle, location, price
- **Special Offers**: Dynamic list of offers
- **Overview**: Detailed project description
- **Key Highlights**: Dynamic list of highlights
- **Pricing Information**: Dynamic pricing table
- **Amenities**: Amenities count
- **Location & Connectivity**: Dynamic location list
- **Developer Information**: Developer description
- **RERA Information**: Registration, advertiser, project number
