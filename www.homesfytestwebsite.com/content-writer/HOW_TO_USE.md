# How to Use Content Writer Dashboard

## Quick Start Guide

### Step 1: Open the Dashboard
Simply double-click `index.html` or drag it into your web browser. The dashboard will open immediately - no installation needed!

### Step 2: Load Sample Data (Optional)
Click the **"üì• Load Sample Data"** button at the top to see a complete example with pre-filled content. This will help you understand how the dashboard works.

### Step 3: Enter Your Content

#### Basic Fields
Fill in the standard fields like:
- Project Name
- Developer Name
- Meta information (Title, Description, Keywords)
- Banner details
- Overview description

#### Dynamic Fields
For sections with ‚ûï (plus) buttons:

1. **Special Offers**: 
   - Default: 1 offer field
   - Click ‚ûï to add more (offer_2, offer_3, etc.)
   - Click ‚úï to remove (but keep at least one)

2. **Key Highlights**:
   - Default: 1 highlight field
   - Click ‚ûï to add unlimited highlights
   - Each gets numbered automatically

3. **Pricing Information**:
   - Default: 1 price entry (Type, Area, Price)
   - Click ‚ûï to add more pricing tiers
   - Example: 2 BHK, 782 sqft, 71 Lac

4. **Location & Connectivity**:
   - Default: 1 location field
   - Click ‚ûï to add more locations
   - Include distance/time information

### Step 4: Customize Filename (Optional)
In the left panel, you can set a custom filename for your exports. If left empty, it will automatically use the project name.

### Step 5: Export Your Content

#### Export to CSV/Excel
1. Click the **"üìä Download CSV/Excel"** button
2. File downloads with format: `filename.csv`
3. Open in Excel, Google Sheets, or any spreadsheet app
4. Format: `Classname (Key),Content`

#### Export to Word Document
1. Click the **"üìÑ Download Word Doc"** button
2. File downloads with format: `filename.doc`
3. Open in Microsoft Word or any word processor
4. Beautifully formatted with sections, tables, and lists

### üìù Best Practices

- **Save your work**: Export regularly to avoid losing content

### File Download Issues
If files don't download:
1. Check browser settings for downloads
2. Allow pop-ups for downloads
3. Check download folder permissions

### Content Not Saving
- All content is client-side only
- Export regularly to save your work
- Refresh will clear unsaved content

### Dynamic Items Not Adding
- Make sure JavaScript is enabled
- Try refreshing the page
- Use a modern browser

## Advanced Usage

### Multiple Projects
- Fill in content for one project
- Export both formats
- Clear fields (refresh page)
- Start next project

### Bulk Data Entry
- Copy-paste content from other sources
- Use "Load Sample Data" as template
- Modify to match your project

### Custom Branding
- All exports use your content only
- Add your logos in Word document manually
- CSV format is data-only
